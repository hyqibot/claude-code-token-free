# Claude Code Token Free v1.0.2

## Changes

- DeepSeek 默认工具链改为 **XML（豆包对齐）**；DSML 保留为可选，设置页选择时会提示可能被风控
- Zero-Token 网关打包路径：源码联调与 sidecar 统一从 `desktop/build-artifacts` 解析 `gateway.bundle.mjs`
- DeepSeek XML 模式补充 URL/下载类任务的【执行要求】，避免模型只回复文字而不调用工具

## Fixes

- 修复打包后 Zero-Token 网关误用 Bun 安装目录作为 app-root 导致 `gateway.bundle.mjs not found`
- 修复 DeepSeek 在 XML 模式下遇到下载链接时不输出 `<tool_call>` 的问题

## 下载说明

- macOS (Apple Silicon)：`.dmg`
- macOS (Intel)：`.dmg`
- Windows：NSIS 安装包（`.exe`）
- Linux：`.deb`
