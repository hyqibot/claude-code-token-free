# Claude Code Free Token v1.0.0

Claude Code Free Token 桌面版

Claude Code 永远免token费，基于 2026 年 3 月 31 日从 Anthropic 的 npm 仓库泄露的源代码修复构建的 Claude Code 版本，添加永久免token费模板，并构建成桌面版，尽情享受！


## 版本包含能力
- 永久免token费的web模型接入 + 工具链能力加强
- Oh-my-claudecode 接入，多智能体编排任务
- 跨平台桌面主程序 + 多标签、多会话管理系统
- Markdown / 代码块 / 代码差异视图优化渲染
- Bash、Read、Edit、Write 等工具调用专属展示面板
- 斜杠命令、`@` 文件检索、附件上传、图片预览
- 多模型服务商配置与资源管理
- Claude 官方账号接入，修复桌面端 OAuth 授权回调
- 定时任务管理面板
- IM 平台适配器（微信/Telegram/飞书）
- 桌面端受控能力配置与系统权限集成
- 完整打通 Windows / macOS / Linux 打包发布流程
