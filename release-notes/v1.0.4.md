# Claude Code Token Free v1.0.4

## Features

- 微信 / 钉钉 IM 出站发图：Agent 生成的本地 PNG 自动上传并以独立图片消息发出
- 桌面聊天页支持会话 transcript 中出现的输出目录（如图表路径）经 `/api/filesystem/file` 加载 PNG

## Fixes

- 修复微信 CDN `upload_full_url` 上传与钉钉 sessionWebhook markdown 发图
- 修复流式 UTF-8 分片导致的路径乱码、同目录多图误匹配与快速连发丢图
- 修复 IM runtime 每条消息重复同步导致第二次生图 AbortError
- IM 模型默认在 Zero-Token 网关不可用时自动 fallback 到其他 Provider
